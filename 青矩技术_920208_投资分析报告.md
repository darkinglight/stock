## 二、行业分析

### 2.1 行业地位

青矩技术是国内首家以工程造价咨询为核心主营业务的上市公司。2025年9月，全资子公司青矩顾问造价咨询收入位列行业第2名，连续十余年稳居全国前三甲，稳居行业领先地位。

公司是国内工程造价咨询行业的龙头企业，深度服务国家建设，项目遍及各个行业和地区——从京沪高铁、北京大兴国际机场、海阳核电站到雄安城市计算中心、小米新能源汽车工厂，覆盖传统基建、新基建、新型城镇化、战略新兴产业。

### 2.2 竞争格局

工程造价咨询行业2024年业务收入约1,112亿元，从业企业超1.6万家，市场极度分散（CR20仅7.81%），头部企业市占率均低于1%。行业龙头为建银咨询（建设银行全资子公司）和青矩顾问两强领跑格局。

二级市场可比的工程咨询上市企业40多家，但多数以工程设计/监理/规划为主营，没有第二家以纯工程造价咨询为核心的公司。

### 2.3 行业趋势

**行业周期位置：成熟期，但格局整合期。**

行业正处于**全咨化、数字化、集中化**三浪叠加阶段：
- **全咨化**：从单一代建造价向全过程工程咨询转型，头部门槛提高
- **数字化/AI化**：AI造价、BIM技术、大数据平台正在改变作业模式
- **集中化**：CR20仅7.81%，行业整合空间巨大，上市公司具备资本并购优势

### 5.4 魔法公式评估

## 七、变化拆解

变化拆解触发：营收增速从2023年+13.36%降至2025年+0.83%，降幅>10pp。

### 营收增长趋缓拆解

```
变化项目：营收增速 13.36% → 2.17% → 0.83%
拆解过程：
- 2023年高增长（+13.36%）：疫情影响消退+基建投资反弹
- 2024年增速骤降（+2.17%）：地方财政紧约束，基建投资放缓，行业整体承压
- 2025年接近停滞（+0.83%）：宏观环境下行，造价咨询行业收入增速同步放缓
- 但扣非净利润增速明显高于营收：2025营收+0.83%，扣非+6.74%——显示内部效率提升
- 在手订单余额较期初增长>10%（2025年报披露）——增长有后劲
持续性判断：周期性放缓，非结构性衰退
```

### 毛利率上升拆解

```
变化项目：毛利率 45.52% → 46.72% → 48.21%
拆解过程：
- 2023→2025毛利率提升2.69pp
- 员工人数下降4.55%但高学历人员占比提升——人效优化
- 标准化流水线+AI辅助降低单位成本
- 产品结构优化：高附加值全过程咨询占比提升
持续性判断：结构性改善可持续
```

### ROE下降拆解

```
变化项目：ROE(加权) 25.53% → 19.74% → 18.47%
杜邦分解：
- 净利率：21.36% → 19.10% → 19.01%（稳定）
- 资产周转率：营收/总资产 = 9.59/15.55(2024) vs 9.67/16.24(2025)（略降）
- 权益乘数：1/(1-35.85%) 1/(1-32.82%)（杠杆下降）
- 核心驱动：权益乘数下降（主动降负债）+ 现金理财增加拖累周转率
持续性判断：ROE中枢从25%向18-20%下移，但高负债率时代不可持续，新ROE水平更健康
```

## 八、三方辩论记录

### Round 1：初始立场

**芒格**（4星）：这是一家好生意。连续10+年行业前三，客户以央企/国企为主，替换成本高，有一定的护城河。轻资产高ROE，无有息负债，管理层持股比例高、利益对齐。但营收增长趋近于零是最大的问号——如果找不到新增长曲线，这只是一个"还不错但不增长"的公司。初始评级：⭐⭐⭐⭐

**格林布拉特**（4星）：ROC 36%在A股名列前茅。Earnings Yield 18.7%极其诱人。PE 9.7x、PB 1.7x、几乎零负债、轻资产模式——魔法公式的两个指标都非常出色。北交所上市导致的折价为主动投资者提供了结构性买入机会。初始评级：⭐⭐⭐⭐

**塔勒布**（4星）：治理评分4.6/5，零质押，ESG AA级——没有明显的尾部风险。但真正的尾部风险不在公司本身，而在宏观：中国固定资产投资增速放缓，如果基建投资大幅下滑，行业可能经历一轮洗牌。公司现金充裕（6.69亿），财务上反脆弱。初始评级：⭐⭐⭐⭐

### Round 2：芒格质询格林布拉特

**芒格：** 你那个18.7%的Earnings Yield确实好看。但问题是——这个Earnings的正常化水平是多少？营收已经快零增长了。如果正常化盈利不是现在的1.89亿，而是1.5亿呢？你的18.7%就变成15%。仍然便宜，但没那么惊艳。还有——这家公司最大的资产是人，人才流失的风险你算进去了没有？

**格林布拉特：** 你说的正常化是关键。我同意——未来盈利可能低于当前。但即使假设未来盈利下降20%（到1.5亿），PE仍然只有12x，Earnings Yield仍然有~15%。对比10年期国债收益率不到3%，这仍然是非常有吸引力的。至于人才风险——你看看他们的数据：员工人数下降4.55%，但人均产值上升了；高学历人员占比提升到88%；平均年龄33.88岁。这不是一个人才在流失的公司，这是一个在优化人力结构的公司。管理层持股比例高，平均任职10年+——这些数据比同行的任何一个都好。

### Round 3：塔勒布质询芒格

**塔勒布：** 你说它"有一定护城河"。什么是这个护城河？告诉我——一个央企客户换掉青矩技术需要多长时间？成本多高？如果明天有另一家咨询公司愿意以半价接活，客户会走吗？我看到的护城河是一个高度分散的行业（CR20仅7.81%），任何一家都没有定价权。你说的"护城河"会不会只是过去的惯性？

**芒格：** 你说得对，它不是可口可乐。但护城河不需要是一堵墙——有时就是一道篱笆。这家公司的护城河是三层的：第一，央企/国企客户的入围门槛需要时间（2025年入围了国家管网、南方电网等多家战略客户，这不是一周能完成的事）。第二，这是以人为主的服务业，客户买的是专业判断和信任记录——信任需要10年建立，可以在5分钟崩塌，但5分钟崩塌的概率很小。第三，唯一以造价咨询为核心的上市公司身份——IPO拿到的品牌溢价是无形资产。但…你问得对，它确实不是不可或缺的。我给它4星而不是5星，就是因为这层原因。

### Round 4：格林布拉特质询塔勒布

**格林布拉特：** 你说的尾部风险具体是什么？"基建投资下滑"这个场景——概率有多少？影响有多大？如果基建投资真的下滑30%，整个行业都会受损，但也意味着存活下来的头部企业会获得更大的市场份额。这家公司有6.69亿现金、零有息负债——它在行业洗牌中可以成为整合者而不是被整合者。按你的杠铃策略，它不就是那个"极度保守"的一边吗？稳定的现金流+充足的现金储备+行业头部地位。

**塔勒布：** 你说到点子上了——我也给了它4星。真正的尾部风险不是基建下滑，而是"应收账款集体坏账"。看看资产负债表：应收账款2.38亿+合同资产4.24亿=6.62亿，占总资产的41%。如果地方政府/城投平台出现系统性违约，这些资产会在一个季度内大幅减值。这是工程咨询行业的结构性脆弱点——你的客户和你的债务人通常是同一个人。我不是说这一定会发生，但这是一个真实的、不对称的尾部风险。好消息是——公司账上6.69亿的现金+理财可以作为缓冲。坏消息是——净现金只有0.07亿（现金+理财-有息负债-应收应付净额），如果加上6.62亿应收/合同资产的风险敞口，这个不对称性需要认真对待。

### Round 5：数据深挖与变化拆解

**深挖结果：**
1. **应收账款账龄：** 2025年报未详细披露账龄结构，但审计报告为无保留意见，中汇所连续7年审计，应收账款/合同资产被列为关键审计事项但未提出重大问题
2. **经营现金流质量：** 2025年经营现金流1.79亿，净利润1.89亿，比例0.95，健康。但需关注合同资产4.24亿转化为现金的节奏
3. **在手订单：** 2025年半年报披露期末在手订单余额29.32亿，为年收入的3倍+，收入可见性强
4. **新签合同：** 2025上半年新签6.58亿，同比+29.50%，需求端在回暖

**变化拆解：**
- 营收增长趋缓：周期性因素为主（地方财政收紧），非结构性衰退
- 扣非净利增速>营收增速：内部效率提升，可持续
- ROE下降：主要来自权益乘数下降（主动去杠杆），是一次性调整

### Round 6：立场调整

**芒格：** 不调整 — 4星。在手订单29.32亿说明增长有支撑，不是零增长的结构性困境。但护城河的天花板确实存在——1,112亿的行业里，头部企业市占率不到1%，说明这个行业本质上是一个难以集中的服务业。这限制了我的评级上限。

**格林布拉特：** 上调至5星。在手订单29.32亿（3x年收入）、新签合同增速29.5%、ROC 36%、Earnings Yield 18.7%、零净负债——这个组合在A股极为罕见。北交所的流动性折价为主动投资者创造了结构性买入机会。

## 九、芒格视角分析

**第一印象：** 这是一个"不增长的好生意"。我喜欢它什么？轻资产、高ROE、管理层持股、零有息负债、稳定的行业地位。我不喜欢什么？营收增长接近零。

**护城河判断：** 三道篱笆——不是墙，不是护城河，是篱笆。
1. 央企/国企客户入围壁垒（每年只能入围有限的新客户）
2. 信任记录和品牌（10+年行业前三的信用资产）
3. 上市公司平台的资本优势（唯一以造价咨询为核心的上市公司）
但这三个篱笆都挡不住价格战——如果竞争对手以半价抢客，客户会不会走？部分会。

**管理层评估：** 董事高管平均任职10年+，持股比例高——激励结构正确。2025年换届将董事会平均年龄从57岁降至45岁，增加AI专家——这是一个在认真管理传承的组织。

**ROIC趋势：** 从32%向36%改善（轻资产模式+效率提升）。这是质的信号——公司不是在"烧钱换增长"，而是在"用效率换利润"。

**逆向思考：** 什么样的情景会让这笔投资亏钱？①增长完全停滞，营收持续性下降；②应收账款/合同资产出现大规模坏账；③核心团队集体出走。三者概率都不高，但①是最大风险。

**放在哪个筐：** Yes（以当前价格）。不是兴奋型的yes，是冷静的yes——这是一个合理的价值投资标的，不是下一个十倍股。

## 十、格林布拉特视角分析

**魔法公式排名：** ROC 36% → 全市场前5%。Earnings Yield 18.7% → 全市场前5%。两个指标同时在前10%的公司极为罕见。

**特殊事件分析：** 2026年3月获批发行不超过4亿元科技创新公司债券——这可能是催化剂的开始。募集资金用于股权投资和补充营运资金，意味着公司可能进行行业并购整合。在一个CR20仅7.81%的极度分散行业里，并购是最有希望的增长路径。

**机构强制卖出分析：** 北交所的流动性折价就是一个典型的"机构被迫不来"场景——大公募由于流动性限制（日均成交额仅~2000-3000万）无法配置北交所标的。这意味着公司被系统性低估，不是因为基本面不好，而是因为很多买家被规则挡在门外。这正是我书中说的"structural opportunity"——市场结构创造了定价错误。

**正常化盈利估算：** 
- 保守场景：假设营收零增长，但毛利率继续改善→正常化净利润~2.0亿
- 中性场景：在手订单29.32亿，年营收10亿+→正常化净利润~2.0-2.1亿
- 乐观场景：4亿科创债落地并购→看情况

以18.36亿市值，中性场景PE ~9x。如果PB回归到2.5x（行业中位），对应股价~20元，上行~46%。

**催化剂：** ①科创债落地后并购（短期）；②北交所流动性改善/转板预期（中期）；③AI平台商业化（长期）。

## 十一、塔勒布视角分析

**尾部风险清单：**
1. **应收账款/合同资产集中违约（中等概率，中等影响）：** 6.62亿应收+合同资产占总资产41%。如果地方政府/城投出现大面积违约，公司可能一次性损失3-5亿——占净资产的30-50%。
2. **核心人才流失（低概率，高影响）：** 人力密集服务业，如果核心团队出走，业务可能断层。
3. **增长停滞变永久（中等概率，中等影响）：** 营收增速从25%→13%→2%→0.8%——不是骤降，是缓降。这可能不是周期性调整，而是结构性天花板。
4. **行业资质门槛取消后的竞争加剧（低概率，低影响）：** 已经发生——2021年取消造价咨询资质要求，但公司地位不降反升。

**反脆弱性评估：**
- **正面冲击**（并购整合成功、AI赋能提效）：上行空间可期
- **负面冲击**（基建投资暴跌、应收坏账）：6.69亿现金+零有息债=有缓冲
- **遍历性检验：** 100家公司里，分散持有10家高ROE低负债的公司，任何一家违约都不会致命。但如果你all-in这一家，应收坏账6.62亿的尾部风险可能是毁灭性的。
- **结论：** 公司本身是鲁棒的（robust），但不是反脆弱的（antifragile）。建议仓位不超过总仓位的10%。

**Skin in the game验证：**
- 管理层持股比例高（核心员工+管理层合计持股~47%）
- 董事高管平均任职10年+
- no one is selling
- ✅ 通过
