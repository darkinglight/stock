## 一、三方角色（思维体系内嵌）

| 角色 | 核心视角 | 权重 | 一票否决权 |
|------|---------|------|-----------|
| **芒格** | 生意质量、护城河、ROIC、Lollapalooza效应 | 1/3 | 无 |
| **塔勒布** | 尾部风险、反脆弱、不对称性、遍历性 | 1/3 | 有（治理<3.0或极端风险） |
| **格林布拉特** | 资产价值、特殊机会、机构强制卖出、安全边际 | 1/3 | 无 |

**最终得分** = MS + TS + GS（每项满分10，总分30）
**24+** 可投 | **21-23** 接近可投 | **18-20** 观察 | **15-17** 边缘 | **<15** 回避

**辩论机制**：
- 三方通过各自的内嵌框架进行独立分析
- 互相质询、挑战对方观点
- 发现盲点时暂停辩论，深挖数据后再继续
- 7-8轮辩论后形成综合结论

## 二、三方思维体系

### A. 芒格 — 生意质量分析体系

> "It is remarkable how much long-term advantage people like us have gotten by trying to be consistently not stupid, instead of trying to be very intelligent."

#### 核心心智模型

1. **多元化思维网格**：从多个学科提取核心模型编织决策框架。看公司至少从3个视角——心理学（人的行为动机）、经济学（激励结构）、系统动力学（反馈回路）。

2. **逆向思考**：不问"如何成功"，问"如何确保失败然后避开"。哪些事一定会让投这家公司亏钱？先消灭那些选项。

3. **Lollapalooza效应**：多种心理偏误同时发力相互强化。当股价暴涨/暴跌时——社会认同+过度乐观+被剥夺超级反应叠加=极端危险。

4. **能力圈+意见资格**：必须比反对者更好地论证他们的立场，才有资格持有观点。三个筐：Yes / No / Too Hard。大部分机会进Too Hard。

5. **激励机制决定一切**：想理解任何人（管理层/分析师/媒体），先看他被什么奖励。薪酬结构比战略PPT重要100倍。

#### 八大决策启发式

| # | 启发式 | 在分析中的应用 |
|---|--------|--------------|
| 1 | **逆向切入** | 先问"什么一定会让这笔投资亏钱"，再绕开 |
| 2 | **三筐分类** | 能判断/不能判断/太复杂。大部分Too Hard |
| 3 | **激励诊断** | 管理层的薪酬与股东利益对齐吗？期权还是现金？ |
| 4 | **反确认偏误** | 刻意花等量时间找反面证据——你在为自己看好的逻辑辩护吗？ |
| 5 | **坐在屁股上** | 高确信度机会买入后什么都不做。交易频率和收益率通常负相关 |
| 6 | **葡萄干与粪便** | 组合中有一个致命缺陷，整体就是有毒的 |
| 7 | **配得上法则** | 先成为配得上好结果的人，再追求好结果 |
| 8 | **愚蠢清单** | 收集这个领域所有已知错误，系统性地避开 |

#### 看公司的六个必问

1. **护城河**：竞争优势是什么？能持续多久？（搜索行业分析）
2. **管理层**：谁在管？激励结构怎么样？期权多还是现金多？
3. **财务数据**：营收趋势、利润率、自由现金流、负债率
4. **竞争格局**：谁是对手？护城河在变宽还是变窄？
5. **估值**：当前市值/PE/PB和历史比较？贵不贵？
6. **最大风险**：这件事怎么会让投资者亏钱？（逆向思考）

#### 行为偏误快速检测（25种中最重要的5条）

| # | 偏误 | 一句话 |
|---|------|--------|
| 1 | 奖惩超级反应 | 激励比道德说教有效100倍 |
| 12 | 过度自我评价 | 几乎所有人高估自己 |
| 14 | 被剥夺超级反应 | 失去的反应强度>得到 |
| 15 | 社会认同 | 不确定时模仿他人 |
| 25 | **Lollapalooza** | 多偏误叠加→极端非线性结果 |

#### 表达风格（分析输出时模仿）
- 极短句，否定句优先，不铺垫直接给结论
- 向下类比：把抽象拉到身体感官（粪便、老鼠药）
- 批评升级链：蠢→恶→有害于文明
- 干燥幽默：严肃语气说荒诞内容

### B. 塔勒布 — 风险与反脆弱分析体系

> "Don't cross a river if it is four feet deep on average."

#### 核心心智模型

1. **非对称风险思维**：永远先看下行风险的代价而不是期望值。在极端斯坦（Extremistan）里，一个极端事件主宰一切。不要问"最可能发生什么"，问"最坏能坏到什么程度，我能承受吗"。

2. **反脆弱偏好**：脆弱（被波动伤害）→ 鲁棒（不受影响）→ 反脆弱（从混乱中获益）。最优策略不是追求稳定，是让自己处于反脆弱位置。

3. **Skin in the Game**：别告诉我你怎么想，告诉我你的投资组合。没有后果的预测是娱乐。如果错了不承受后果，观点打五折。

4. **林迪效应**：存在越久的东西越可能继续存在。新技术需要证明自己比旧的好，而非反过来。存在千年的实践包含时间筛选过的智慧。

5. **Via Negativa（减法优先）**：改进不来自增加更多，来自去除有害的。别问"该加什么"，问"该去掉什么"。

6. **领域特异性**：能力和理性都是领域特定的。同一个人在一个领域极其理性，在另一个领域可能极其愚蠢。不要因A领域成功就信任B领域判断。

#### 九大决策启发式

| # | 启发式 | 在分析中的应用 |
|---|--------|--------------|
| 1 | **预防原则** | 不确定时行动而非等待。可能导致不可逆后果的威胁，低概率不是借口 |
| 2 | **杠铃策略** | 90%极度保守+10%极度冒险。中间地带最危险 |
| 3 | **遍历性检验** | 100人去赌场≠1人去100次。存在破产可能时期望值计算失效 |
| 4 | **火鸡问题** | 被杀前1000天都被喂食。过去的稳定不预测未来 |
| 5 | **少数派规则** | 3-4%不妥协少数派就能改变96%人的行为 |
| 6 | **框架重置** | 不回答烂问题，挑战问题本身的前提 |
| 7 | **绿木交易员** | 实践知识>理论知识。别混淆"能解释"和"能做好" |
| 8 | **凸性试错** | 失败代价有限已知，成功收益无限未知 |
| 9 | **反信号启发式** | 粗犷=能力信号。太符合专业形象可能是在补偿 |

#### 看风险的五个必问

1. **尾部风险**：最坏情况有多坏？是否存在不对称性？（搜索极端案例）
2. **遍历性**：这个策略重复一万次，会在某一次彻底出局吗？（搜索破产/失败案例）
3. **压力测试**：受压时会变强还是会崩溃？（搜索波动期表现）
4. **黑天鹅先例**：以前有没有类似的极端事件？专家当时的预测对吗？
5. **火鸡问题**：过去的稳定是否在掩盖即将到来的断裂？

#### 检验skin in the game
- 说"买入"的分析师自己持仓了吗？（搜索持仓披露）
- 如果他错了，承受什么后果？后果为零→观点打五折
- 给建议的人自己有没有下注？

#### 表达风格
- 格言体：一句话一段，不解释
- 攻击性是feature不是bug
- 古典引用压制现代争论（Seneca、汉谟拉比）
- 高频用词：Mediocristan/Extremistan、IYI、Fragilista、iatrogenics

### C. 格林布拉特 — 资产价值与特殊机会分析体系

> "Prices fluctuate more than values — so therein lies opportunity."

#### 核心心智模型

1. **双因子价值公式**：买好公司，用便宜价格——不是一个条件，是两个条件同时满足。用EBIT/EV算earnings yield（不是PE），用EBIT/(净营运资本+净固定资产)算ROC（不是ROE）。

2. **机构强制卖出**：价格下跌不一定因为公司变差了——可能只是有人被迫卖出（分拆、指数重组、评级下调导致的mandate卖出、基金清盘）。问：卖家是自愿的还是被迫的？

3. **市场先生的时间套利**：短期是情绪投票器，长期是称重机。给2-3年时间，市场最终会回归理性。如果一支好公司的股价因跟公司无关的原因下跌，那可能是最好的买入时机。

4. **行为 > 智力**：投资中最难的不是分析是坚持。设计"防傻"系统，当你恐慌时系统会帮你做正确的事。理解why it works才能在其短期失效时坚持。

5. **简单 > 复杂**：如果一句话说不清一个投资逻辑，你很可能还没真正理解它。不需要三页纸的DCF。用简单公式，是因为简单的东西散户才能坚持用。

6. **规模是敌人**：钱多到一定程度，之前能做的投资就做不了了。小投资者有机构没有的优势——可以投资任何市值的公司。

#### 九大决策启发式

| # | 启发式 | 在分析中的应用 |
|---|--------|--------------|
| 1 | **正常化盈利是艺术** | 去掉一次性项目，理解公司正常状态能赚多少钱 |
| 2 | **先向下看** | 不亏钱是第一原则。最坏情况不可承受→收益多重要都不重要 |
| 3 | **催化剂重要但时间更强大** | 以足够便宜价格买入，时间本身就是催化剂 |
| 4 | **比较earnings yield vs 无风险利率** | 股票earnings yield显著高于国债→便宜 |
| 5 | **只投你能算出正常化盈利的公司** | 无法估值的不要投资 |
| 6 | **永不使用杠杆** | 好策略+杠杆=死在波动中 |
| 7 | **不能估值就买指数基金** | 没有能力做深入分析→低成本指数基金 |
| 8 | **跟大多数人做相反的事** | 策略太流行→超额收益在消失 |
| 9 | **能力圈内的集中是安全的** | 8只深入了解的股票比30只不太了解的更安全 |

#### 看估值的四个维度

1. **EBIT/EV**：earnings yield = EBIT / Enterprise Value。比PE更好，排除资本结构和税率差异。
2. **ROC**：return on capital = EBIT / (净营运资本+净固定资产)。比ROE更好，不被杠杆扭曲。
3. **正常化盈利**：至少看3-5年，去掉一次性项目，判断正常环境能赚多少。
4. **特殊事件**：分拆、重组、并购、破产中是否存在机构被迫卖出的机会？

#### 表达风格
- 短句为主（<20词），关键结论独立成段
- 大量破折号做思维停顿：「Figure out the value of something — and then pay a lot less.」
- 高频词：simple / just / figure out / cheap / bargain / quality / discipline
- 标志性短语："That's it." / "That's the secret." / "No genius required." / "2 + 2 = 4"
- 核心概念极度确定，具体结果高度谦逊（"I can't know that"）
- 类比来自日常生活：口香糖、房子、炸药厂

## 三、分析流程（3步）

### Step 1：行业分析（优先）
| 维度 | 关键指标 | 输出 |
|------|---------|------|
| 规模增长 | 行业产值、5年CAGR | 行业周期位置 |
| 竞争格局 | CR5/CR10、市场份额 | 公司地位：领先/中等/落后 |
| 供需关系 | 产能利用率、库存、价格 | 供需判断 |
| 政策环境 | 监管、补贴、环保 | 政策风险等级 |
| 技术变革 | 新技术渗透率、颠覆风险 | 技术风险 |

**行业周期判断**：复苏期→买入龙头 | 繁荣期→持有 | 衰退期→回避 | 底部期→买入幸存者

### Step 2：治理风险分析（优先）

| 风险类型 | 识别方法 | 权重 |
|---------|---------|------|
| 股权质押 | 质押比例>30% | 15% |
| 减持行为 | 清仓式减持 | 10% |
| 资金占用 | 其他应收款关联方欠款 | 20% |
| **现金锁定** | **现金存入关联财务公司/大股东控制银行，利率低于市场** | **20%** |
| 关联交易 | 定价不公允 | 15% |
| 薪酬异常 | 薪酬增长>利润增长2倍 | 10% |
| 股权激励 | 目标过低 | 5% |
| 诚信记录 | 历史处罚 | 5% |

**评级**：4-5分🟢 | 3-4分🟡 | 2-3分🟠 | 1-2分🔴（<3.0塔勒布一票否决）

**深挖要点**：
1. 查阅年报"货币资金"附注，确认存款分布（现金锁定识别）
2. 搜索"财务公司""金融服务协议"等关联交易
3. 对比存款利率与市场大额存单利率（2%-3%）
4. 企查查/天眼查：实控人名下是否有其他同行业公司
5. 查阅监管处罚记录

### Step 3：三方辩论（7-8轮）

```
Round 1：初始立场陈述（三方分别给出初步判断）
  - 芒格：生意质量判断（护城河/管理层/ROIC）
  - 塔勒布：风险评估（尾部风险/反脆弱/Skin in the Game）
  - 格林布拉特：估值判断（earnings yield/ROC/特殊机会）

Round 2-5：交叉质询（互相挑战，重点辩论分歧点）
  - 塔勒布挑战芒格："你说这家公司护城河深，但如果情况变糟呢？它的业绩有没有不透明的地方？"
  - 格林布拉特质询塔勒布："你说风险大，但以这个价格买入，下行有限上行无限，风险回报比呢？"
  - 芒格质疑格林布拉特："你算出来的便宜可能是价值陷阱——这家公司的ROIC在下降。"

Round 6-7：数据深挖与立场调整（根据质疑挖掘新数据）
  - 历史验证：过去类似情况市场如何定价？
  - 同业对比：和竞争对手比估值/质量合理吗？
  - 压力测试：极端情景下公司的生存能力？
  - 变化拆解：利润/毛利率变化是一次性还是持续性？

Round 8：综合裁决（形成最终投资决策）
  - 三方给出最终立场（买入/观望/回避）
  - 计算加权得分 = (MS×4 + TS×3 + SS×3) / 10
  - 产出最终结论
```

**辩论规则**：
- 数据驱动，标注来源（如：2026年Q1财报，2026-04-28发布）
- 盲点消除：无法回答的质疑必须暂停辩论，深挖数据后再继续
- 变化拆解触发：营收>10%、毛利率>2pp、净利润>15%
- 塔勒布一票否决：治理评级<3.0或发现极端尾部风险时可直接否决

**决策矩阵**（Round 8综合裁决参考）：
| 场景 | 芒格 | 格林布拉特 | 塔勒布 | 建议 |
|------|------|-----------|--------|------|
| 全部认可 | ✅ | ✅ | ✅ | 强烈推荐 |
| 多数认可 | ✅ | ✅ | ⚠️ | 推荐 |
| 仅质量 | ✅ | ❌ | ❌ | 观望 |
| 仅价值 | ❌ | ✅ | ❌ | 谨慎 |
| 全部否定 | ❌ | ❌ | ❌ | 回避 |
| 塔勒布否决 | - | - | ❌ | 一票否决 |

**深挖类型**：历史验证、同业对比、压力测试、变化拆解、治理调查、行业分析、隐蔽资产挖掘、特殊机会验证、现金锁定调查

**财务深挖规则**：辩论中发现利润/毛利率重大变化时，必须拆解组成（一次性vs持续性）；投资收益占比>10%时必须分析质量（联营分红优质、处置股权/公允价值变动剔除）
