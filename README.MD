# INSTALL

## ENV PREPARE
1. python -m pip install briefcase
2. briefcase new .

## DEVELOP
1. briefcase dev

## BUILD EXE
1. briefcase create 
2. briefcase build 
3. briefcase run 

## BUILD INSTALLER
1. briefcase package --adhoc-sign

## BUILD ANDROID
1. briefcase create/update android 
2. briefcase build android 
3. briefcase run android
