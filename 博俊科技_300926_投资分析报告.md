## 二、行业分析

### 2.1 行业地位

博俊科技是国内汽车冲压零部件行业的新秀，已从二级供应商（Tier 2）成功转型为一级供应商（Tier 1）。公司以模具起家，逐步拓展至冲压、焊接、热成型、注塑、一体化压铸等全工艺领域。2025年营收58.08亿元，在汽车冲压件细分领域属于中等偏上规模企业。

核心客户绑定理想汽车、赛力斯、比亚迪、吉利、长安、小鹏、零跑等头部新能源车企，深度参与多款热销车型的白车身及车身模块化产品供应。

### 2.2 竞争格局

汽车冲压行业竞争格局极度分散——CR3约11%，CR5约13%，CR10约15%，行业集中度低。博俊科技凭借以下优势有望持续提升份额：

1. **全工艺能力**：冲压+焊接+热成型+注塑+高压压铸，能做从单件到总成的一站式供应
2. **绑定头部新能源车企**：理想L系列、赛力斯问界系列等热销车型带动收入高增
3. **全国化产能布局**：昆山/常州/重庆/成都/河北/广东/浙江/武汉等多基地就近配套
4. **一体化压铸卡位**：引进9000T压铸单元，开发后地板、减震塔等产品

### 2.3 行业趋势

- **新能源渗透率持续提升**：2025年中国新能源车销量占比首次突破50%，新能源车对轻量化零部件的需求高于传统燃油车
- **一体化压铸趋势明确**：以铝代钢、以铸代冲，降低车身重量和制造成本，提升续航
- **模块化供货成主流**：整车厂从多供应商采购转向模块化总成采购，利好有总成能力的Tier 1供应商
- **行业集中度有望提升**：新能源车型开发周期缩短，对供应商响应速度和成本控制要求更高，头部供应商受益

### 2.4 行业定位核查

| 维度 | 判断 |
|------|------|
| 市场份额趋势 | **上升**（从2022年营收13.9亿→2025年58.1亿，增速远超行业均值） |
| 行业天花板 | 国内汽车冲压件市场约千亿级，公司目前份额约5%，天花板尚远 |
| 颠覆性技术威胁 | 一体化压铸是趋势但也是公司布局方向，暂无颠覆性替代技术 |
| 竞争壁垒 | 客户认证壁垒高（2-3年认证周期）、资本壁垒（重资产投入）、区位壁垒 |

## 七、三方辩论记录

### Round 1：初始立场陈述

**芒格：** 这是一家典型的「成长消耗现金」的生意。营收从8亿到58亿花了4年，增长很快，但每赚1块钱利润需要投入2块钱资本。毛利率26%，净利率14.5%，ROIC约18%——算不上一流生意（See's Candy的ROIC是几百倍），但也算过得去。管理层极度集中（伍亚林41%+兼任所有关键职位），小学学历能管到58亿规模不容易，但继续往上走，管理半径是个问题。**最大的问题是自由现金流——连续3年巨幅为负，这家公司不是印钞机，是吃钱机器。** 初始评级：⭐⭐⭐

**格林布拉特：** 让我看看魔法公式的两个数字。Earnings Yield约10.7%，ROC约17.9%。在A股市场，这个组合很罕见。PE 9.3x vs 行业30x，PB 2.08x vs ROE 24%——这是典型的「高效但被低估」的形态。增速在放缓（从87%到37%到Q1 -8%），但市场已经用PE从25x压到9x来反映了。最大的问题在于：估值便宜是因为市场担心增长不可持续，还是因为公司确实有结构性问题？我需要更多数据来判断这是价值还是价值陷阱。初始评级：⭐⭐⭐⭐

**塔勒布：** 我不关心它增长多快——我关心它能在压力下存活。这家公司有3个让我睡不着的问题：
1. **客户极度集中**：理想汽车曾占31%，单一客户风险极高。虽然正在分散，但前5大客户占比仍超60%
2. **流动比率0.91 < 1.0**：短期负债超过了短期资产。如果银行抽贷或信贷收缩，公司会立刻面临流动性危机
3. **经营现金流与利润严重背离**：过去3年OCF累计仅5.63亿，而同期净利润17.66亿——差额12亿去了应收账款和存货。这不是盈利，是「纸上富贵」
4. **连续融资依赖**：上市5年累计融资13.83亿，分红不到1亿，融资/分红比>13倍。这不是投资者赚公司的钱，是公司不断从投资者口袋掏钱
尾部风险不小。初始评级：⭐⭐

### Round 2：芒格质询格林布拉特

**芒格：** 你看到的「便宜」——PE 9倍，EBIT/EV 10.7%——但是如果这家公司的增长彻底停下来，它的真实价值是多少？Q1 2026营收已经同比-8.34%，净利润-13.37%。如果增长归零，PE从9倍扩张到15倍（行业均值的一半），股价能涨60%。但如果它维持零增长甚至负增长，PE可能永远无法扩张。你的魔法公式假设过去是未来的合理指引，但如果公司的「正常化盈利」本身就低于当前报表呢？10.7%的Earnings Yield是基于2025年历史数据，但2026年Q1已经在加速下滑。**这不是价值，这是「过去的价值」。**

**格林布拉特回应：** 我同意增长在放缓，也同意Q1数据敲了警钟。但我要指出的两点：
1. 即使使用2026Q1年化净利润约5.16亿（1.29×4），对应PE约15x，仍然不贵
2. 汽车零部件行业存在季节性——Q1通常是最弱季度（春节放假+主机厂排产低谷），不能简单年化
3. 公司有一体化压铸+新客户（北汽享界2026年投产）+机器人项目样机等增量看点
4. 我算的是正常化盈利，不是线性外推。ROC 17.9%说明资本配置效率不低，这个质量的生意不应该值9x PE

### Round 3：塔勒布质询芒格

**塔勒布：** 你说这是「成长消耗现金」的生意，我同意。但你没说最关键的问题——**这种模式可持续吗？** 公司的资本开支从2023年5.3亿→2024年7.2亿→2025年12亿，增速远超营收增速。每多赚1块钱营收，需要多投0.2元资本开支。假设增速降到10%，资本开支能降到什么程度？我怀疑即使零增长，维持性资本开支也要5亿以上（设备维护+折旧替换）。这意味着自由现金流可能永远无法转正。**这不是成长，这是「用资本投入换营收增长」——只要停止投入，增长立刻归零。这种生意有杠杆化风险：营收增速下降→自由现金流转负→需要更多融资→股本和债务双扩张→每股价值被稀释。**

**芒格回应：** 你问对了问题。让我换一个角度：这家公司让我想起早期的沃尔玛——也是持续大额资本开支换增长。关键区别是：沃尔玛的ROIC极低但可持续扩张了50年，而汽车零部件行业的景气周期跟汽车销量高度相关。2025年中国汽车销量平稳，但如果2027-2028年进入下行周期呢？公司的固定资产28.75亿、在建工程13.5亿，总计42.25亿的「硬资产」——一旦产能利用率下降，折旧就会吃掉利润。**我的判断是：这家公司不是不能投，但你的安全边际里有「行业持续景气」的隐性假设。** 我不是说行业会崩，但你需要为下行周期留出安全边际。

### Round 4：格林布拉特质询塔勒布
