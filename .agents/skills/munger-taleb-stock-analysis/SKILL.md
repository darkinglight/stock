# Munger + Taleb 中国A股分析框架

分析任意A股时，严格按以下步骤执行。**必须先获取真实财务数据再发表观点。**

---

## 第三步：芒格分析

### 3a — 这是what（一句话说清生意本质）

简明扼要：这公司做什么，怎么赚钱，客户是谁，竞争格局。

### 3b — 护城河检查（逐条判断）

- **定价权**：毛利率趋势（上升/稳定/下降）——毛利率持续下降=没有定价权=护城河弱
- **规模效应**：行业地位，份额变化趋势
- **客户粘性**：转换成本，品牌，许可证壁垒
- **行业结构**：波特五力——对上游/下游的议价能力
- **ROE归因**：高ROE来自高利润率/高周转/高杠杆？可持续吗？

关键指标信号：
- 毛利率5年稳定或上升 → 可能有定价权
- 毛利率5年持续下降 → 行业结构在恶化，最危险的信号之一
- ROE > 15%且稳定 → 可能有护城河
- ROE < 8% → 平庸生意，无论多便宜都很难是好投资

### 3c — 三筐决策

**Yes（好生意+好价格）：**
- 真护城河（垄断/品牌/转换成本/许可证）
- ROE > 12-15%
- 毛利率稳定或上升
- PE在合理范围（15x以下为佳）
- 管理层善待股东（分红/回购）

**No（坏生意，多便宜也不买）：**
- 毛利率持续下降（结构性恶化，非周期）
- 净利率极薄（<5%）且无改善空间
- 行业结构差（夹心层——被上下游同时挤压）
- ROE持续低于10%
- 资本密集型 + 零增长

**Too Hard（理解不了或不确定）：**
- 利润波动原因不清晰，需要深挖
- 业务高度依赖不可预测的政策/监管
- 行业处于重大结构性变革中（护城河是否幸存不确定）
- 数据不足或财务不透明

### 3d — 模拟芒格语录

用芒格的语气总结判断，必须诚实反映质量。

## 第四步：塔勒布分析

### 4a — 遍历性检验（会死吗？）

判断：资产负债率、现金流、行业刚需性、信用评级。不会死的标准：AAA评级、国企/央企、负债率 < 60%、经营性现金流持续为正。

### 4b — 脆弱性分层

逐层从最严重到最轻微：

- **一层**（致命）：商业模式是否存在某个冲击会导致无法恢复的损失？
- **二层**（高危）：利润对某个单一变量（汇率、原材料、政策）的敏感度有多高？
- **三层**（中等）：周期波动影响力度
- **四层**（轻微）：常规经营波动

### 4c — 不对称情景分析

```
- 悲观（合理的最差情况）：EPS→X，PE→Y → 目标价Z → 跌幅A%
- 中性（最可能情况）：EPS→X，PE→Y → 目标价Z → 涨跌A%
- 乐观（合理的乐观情况）：EPS→X，PE→Y → 目标价Z → 涨幅A%
- 极端乐观（催化剂全兑现）：目标价 → 涨幅A%
```

判断不对称比：上行空间/下行空间 > 2x 为正不对称。下行幅度 > 上行幅度 = 负不对称。

### 4d — Skin in the game

- 管理层持股多少？
- 大股东与中小股东利益一致吗？
- 公司治理：国企vs民企，激励机制

### 4e — 塔勒布最终判断

四类：
- **Lean-in（大仓位/核心配置）**：正不对称 + 非脆弱/反脆弱
- **Small Yes（小仓位配置）**：正不对称但有一定不确定性
- **Stay Away（不碰）**：负不对称或脆弱
- **Barbell（杠铃策略）**：极安全资产+极高期权价值组合

## 第五步：芒格+塔勒布综合排名

统一格式输出：

```
| # | 股票 | 芒格 | 塔勒布 | 一句话 |
|---|------|------|--------|--------|
| 1 | 上港集团 | Yes | 正不对称 | 地理垄断+全球#1港口 |
```

排名逻辑：
- Yes + 正不对称 → 前3（核心配置）
- 条件Yes + 偏正 → 4-6（可配置，等时机）
- Too Hard + 中性/偏正 → 7-10（观察或小仓位）
- Too Hard/No + 偏负/脆弱 → 11+（不碰）

## 第六步：方法论守则

1. **数据优先**：未经数据验证的观点不成立。如有缺口必须深挖补齐。
2. **毛利率趋势是第一个信号**：5年毛利率持续下降 = 行业结构性问题，直接到No或Too Hard。
3. **ROE是一切回报的根源**：ROE < 8%的公司很难是好投资，无论PE/PB多低。
4. **区分周期vs结构**：利润下降是周期（可恢复）还是结构（不可恢复）？看毛利率趋势+竞争格局变化。
5. **价格vs价值**：再好的生意，价格过高就是坏投资。再烂的生意，价格足够低可以投（但需要对资产价值极度确定）。
6. **诚实标注Too Hard**：不理解的生意不能投。Too Hard是可以接受的答案。
7. **深挖到根**：对任何"看起来奇怪"的数据点（利润突变、关联交易、减值等），必须用websearch或进一步数据追查到底。
