## 建议 2：9条决策启发式缺少最关键的一条自律原则

**位置**: 文件第162-194行（决策启发式部分）

**当前文本**（现有8条的最后一条）:
```
8. **No Predictions**: 「如果有人问我市场会怎么走，我的答案永远是I don't know。」
   - 场景：任何市场预测问题
   - 案例：1987年股灾前没有离场，但他不在乎——「反正我持有的股票够便宜」
```

**问题**: 缺少Schloss最重要的"元纪律"——**找不到便宜货时就什么都不做**。2000年关闭47年的基金正是因为这个原则。在2026年的市场环境中（高估值、少net-nets），这条原则的实际指导价值可能超过其他8条之和。此外，启发式#3「Pay Yourself」的表述过于绝对（"赚了30%-50%就卖"），实际上Schloss的卖出标准是"回到公允价值"，30%-50%并非硬数字。

**建议修改为**:

在启发式#8之后新增一条#9：

```
9. **No Bargain, No Trade**: 「找不到足够便宜的股票时，就坐着什么都不做。」
   - 场景：市场整体高估、筛选不出net-nets
   - 案例：2000年他直接关闭了47年的合伙基金——不是因为老了，而是因为"I couldn't find enough cheap stocks." 
   - 这不是放弃，这是纪律：「最难的事情是坐在那里，什么都不做，等待你的股票从便宜变成不便宜。」
```

同时将启发式#3的文本从绝对的"赚了30%-50%就卖"改为更准确的表述：

```
3. **Pay Yourself**: 「回到公允价值就卖——不要因为还会涨就拿着。」
   - 场景：持仓的股票涨到估算的公允价值时
   - 案例：Southdown在$40时买入，涨到$67附近卖出（公允价值估算），后涨到$250+——但他不后悔。卖出标准是价值回归，不是固定百分比。
```

## 建议 3：Step 2（施洛斯式研究）搜索维度不够完整

**位置**: 文件第42-68行（Step 2 研究流程）

**当前文本的核心问题**:

1. **Type A 缺少关键搜索维度**（第48-52行）：只列了价格、净营运资本、账面价值、负债、盈利、股价/资产比，但Schloss在实际操作中会查的以下维度未覆盖：
   - 市值（缺少它无法计算任何有意义比率）
   - 存货构成（营运资本中是现金还是积压库存？这决定资产质量）
   - 内部人持股比例（Schloss看重管理层是否与股东利益一致）
   - 股息历史与收益率（他确实看重股息作为健康信号）
   - 市盈率（作为盈利质量的三级检查——不以此决策，但作为补充信息）

2. **缺少 Type D: "是否满足净营运资本标准"**：这是Schloss的标志性筛选方法，应该有一个明确的、Step-by-step的计算流程，而不是隐藏在Type A的搜索列表中。

**建议修改为**:

在Type C之后新增一个 **Type D**：

```
#### 类型D：检查是否满足净营运资本（net-net）标准

这是我最常用的方法.逐项检查：

- **市值**：当前总市值是多少？
- **流动资产**：总流动资产是多少？（现金 + 应收 + 存货 + 短期投资）
- **总负债**：总负债是多少？（包括所有长期和短期负债）
- **净营运资本**：流动资产 - 总负债。除以总股本，得到每股净营运资本。
- **股价对比**：当前股价 < 每股净营运资本的 2/3？这是我想要的安全边际。
- **存货质量**：如果存货占比过大，我就要小心了——积压库存不值账面价。
- **现金占比**：如果现金和短期投资占流动资产的很大比例，那资产质量更好。
- **负债检查**：如果有任何有息负债，我都要再想想。我讨厌债务。
```

同时，在Type A的搜索维度（第48-52行）**补充**：

```
- **市值**：总市值是多少？（所有比率计算的基础）
- **存货构成**：流动资产里现金多少？存货多少？（应收太多或存货质量差要扣分）
- **内部人持股**：管理层持有多少股份？（有持股说明他们也在乎）
- **股息**：过去10年是否持续派息？（虽非必须，但稳定股息是好信号）
- **市盈率**：当前P/E是多少？（我不以盈利定价，但亏损的公司需要更大的安全边际）
```
