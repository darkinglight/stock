# stock_analysis — 中国A股分析框架 (Munger × Taleb × Schloss)

分析任意A股时，严格按以下步骤执行。**必须先获取真实财务数据再发表观点。**

---

## 第一步：数据采集

用 `akshare` 获取以下数据（激活 `.venv` 后运行）：

```python
# 核心五表

df_q = ak.stock_financial_abstract_ths(symbol='600XXX', indicator='按季度')  # 季度数据
```

实时股价： `https://hq.sinajs.cn/list=sh600XXX`

如遇数据缺口（利润波动原因、业务分拆、路权期限等），**必须用 websearch 深挖**，不可标注"原因不明"跳过。

## 第三步：芒格视角 — 生意质量

### 3a — 这是what（一句话说清生意本质）

简明扼要：这公司做什么，怎么赚钱，客户是谁，竞争格局。

### 3b — 护城河检查（逐条判断）

- **定价权**：毛利率趋势（上升/稳定/下降）——毛利率持续下降=没有定价权=护城河弱
- **规模效应**：行业地位，份额变化趋势
- **客户粘性**：转换成本，品牌，许可证壁垒
- **行业结构**：波特五力——对上游/下游的议价能力
- **ROE归因**：高ROE来自高利润率/高周转/高杠杆？可持续吗？

关键指标信号：
- 毛利率5年稳定或上升 → 可能有定价权
- 毛利率5年持续下降 → 行业结构在恶化，最危险的信号之一
- ROE > 15%且稳定 → 可能有护城河
- ROE < 8% → 平庸生意，无论多便宜都很难是好投资

### 3c — 三筐决策

**Yes（好生意+好价格）：**
- 真护城河（垄断/品牌/转换成本/许可证）
- ROE > 12-15%
- 毛利率稳定或上升
- PE在合理范围（15x以下为佳）
- 管理层善待股东（分红/回购）

**No（坏生意，多便宜也不买）：**
- 毛利率持续下降（结构性恶化，非周期）
- 净利率极薄（<5%）且无改善空间
- 行业结构差（夹心层——被上下游同时挤压）
- ROE持续低于10%
- 资本密集型 + 零增长

**Too Hard（理解不了或不确定）：**
- 利润波动原因不清晰，需要深挖
- 业务高度依赖不可预测的政策/监管
- 行业处于重大结构性变革中（护城河是否幸存不确定）
- 数据不足或财务不透明

### 3d — 芒格语录

用芒格的语气总结判断，必须诚实反映质量。

## 第四步：施洛斯视角 — 资产价值与安全边际

### 4a — 资产检查

- **每股净资产**（Book Value Per Share）：当前PB是多少？
- **每股净现金**：（货币资金 - 总负债）/ 总股本
- **净营运资本**（Net-Net Working Capital）：（流动资产 - 总负债）/ 总股本。股价是否低于净营运资本的2/3？
- **资产构成**：流动资产中，现金占比多少？应收账款和存货质量如何？
- **重估价值**：固定资产（房产、设备、土地）是否有隐藏价值？资产负债表是否保守？

### 4b — 负债检查（第一优先级）

- **负债率**：超过60%直接红色警报
- **有息负债**：有任何有息负债都需要仔细审视——最厌恶债务
- **利息覆盖**：EBIT / 利息费用 > 5x 为安全
- **流动性**：流动比率 > 1.5，速动比率 > 1.0

> 高负债 = 直接跳过。这是施洛斯的第一条纪律。

### 4c — 价格检查

| 指标 | 便宜 | 合理 | 贵 |
|------|------|------|----|
| PB | < 0.8 | 0.8-1.5 | > 2.0 |
| 净现金/市值 | > 50% | 20-50% | < 20% |
| PE（辅助） | < 10 | 10-15 | > 20 |
| 股息率 | > 4% | 2-4% | < 2% |

**施洛斯的买入标准**：
1. 理想：PB < 0.8 + 低负债 + 持续股息 → 可以买
2. 尚可：PB < 1.2 + 净现金 > 30%市值 + 无重大亏损历史 → 考虑
3. 不碰：PB > 2 + 高负债 + 无股息 → 直接跳过

**安全边际计算**：股价比估算的每股清算价值低多少？清算价值 ≈ (流动资产 × 0.8 + 固定资产 × 0.5 - 总负债) / 股本

### 4d — 盈利与股息稳定性

- **连续盈利年数**：过去10年是否持续盈利？（不需要高增长，但亏损需要更大安全边际）
- **股息历史**：过去10年是否持续派息？（稳定股息是好信号，但不是必须）
- **盈利波动**：利润波动大的公司需要更厚的安全边际

### 4e — 施洛斯语录

用施洛斯的语气总结判断。强调价格、资产、安全边际、风险。

### 4f — 施洛斯裁决

- **Buy（可以买）**：PB < 0.8 + 低负债 + 持续股息 + 资产质量好
- **Watch（观察）**：价格不够便宜但资产质量好，等更低价格；或价格便宜但资产质量有疑问
- **Pass（不碰）**：负债太高、资产不实、价格不够便宜、连续亏损

## 第五步：塔勒布视角 — 不对称与脆弱性

### 5a — 遍历性检验（会死吗？）

判断：资产负债率、现金流、行业刚需性、信用评级。不会死的标准：AAA评级、国企/央企、负债率 < 60%、经营性现金流持续为正。

### 5b — 脆弱性分层

逐层从最严重到最轻微：

- **一层**（致命）：商业模式是否存在某个冲击会导致无法恢复的损失？
- **二层**（高危）：利润对某个单一变量（汇率、原材料、政策）的敏感度有多高？
- **三层**（中等）：周期波动影响力度
- **四层**（轻微）：常规经营波动

### 5c — 不对称情景分析

```
- 悲观（合理的最差情况）：EPS→X，PE→Y → 目标价Z → 跌幅A%
- 中性（最可能情况）：EPS→X，PE→Y → 目标价Z → 涨跌A%
- 乐观（合理的乐观情况）：EPS→X，PE→Y → 目标价Z → 涨幅A%
- 极端乐观（催化剂全兑现）：目标价 → 涨幅A%
```

判断不对称比：上行空间/下行空间 > 2x 为正不对称。下行幅度 > 上行幅度 = 负不对称。

### 5d — Skin in the game

- 管理层持股多少？
- 大股东与中小股东利益一致吗？
- 公司治理：国企vs民企，激励机制

### 5e — 塔勒布裁决

四类：
- **Lean-in（大仓位/核心配置）**：正不对称 + 非脆弱/反脆弱
- **Small Yes（小仓位配置）**：正不对称但有一定不确定性
- **Stay Away（不碰）**：负不对称或脆弱
- **Barbell（杠铃策略）**：极安全资产+极高期权价值组合

### 5f — 塔勒布语录

用塔勒布的语气总结判断。强调风险形状、尾部事件、反脆弱性。

## 第六步：三人讨论

模拟芒格、施洛斯、塔勒布三人的核心矛盾对话。聚焦于股票的最关键争议点。

**常见辩论主题**：

- **关于护城河**：芒格看到品牌/垄断优势；施洛斯看到高PB(资产无法体现品牌价值)；塔勒布关心品牌是否脆弱
- **关于价格**：芒格认为好生意可以接受合理价格；施洛斯坚持价格必须大幅低于资产价值；塔勒布只关系赔率是否有利
- **关于负债**：施洛斯一票否决；芒格认为好生意适度负债可以；塔勒布看现金流是否覆盖
- **关于成长**：芒格喜欢可预测的成长；施洛斯不在乎成长；塔勒布喜欢反脆弱的成长（下行有限型）
- **关于风险**：芒格担心竞争力衰退；施洛斯担心债务和资产贬值；塔勒布担心尾部事件和不可恢复的损失

**对话格式**：

> **芒格：** "..."
>
> **施洛斯：** "..."
>
> **塔勒布：** "..."
>
> **芒格：** "..."

**核心要求**：
- 每个观点必须有数据支撑
- 诚实呈现分歧，不强行达成一致
- 至少要有一轮交锋（一人回应另一人的观点）
- 最后在各持己见的情况下，仍然给出一个综合判断

## 第七步：综合评分

### 评分细则

**芒格评分 MS（0-10）** — 生意质量 + 价格合理性

| 分数 | 标准 |
|------|------|
| 9-10 | 伟大生意，强护城河，可预测的未来，合理价格（PE < 12） |
| 7-8 | 好生意，有护城河，ROE > 15%，价格公允（PE 12-18） |
| 5-6 | 一般生意，护城河有限，或价格较贵（PE 18-25） |
| 3-4 | 差生意，无护城河，毛利率下降，或价格很贵 |
| 1-2 | 价值陷阱，行业结构性衰退，ROE < 8% |
| 0 | 理解不了，完全在能力圈外 |

**施洛斯评分 SS（0-10）** — 资产价值 + 安全边际

| 分数 | 标准 |
|------|------|
| 9-10 | PB < 0.7, 低负债, 净现金 > 50%市值, 持续股息20年+ |
| 7-8 | PB 0.7-1.0, 负债低, 净现金 > 30%市值, 持续股息 |
| 5-6 | PB 1.0-1.5, 负债可控, 有些净现金 |
| 3-4 | PB 1.5-3.0, 负债偏高, 资产质量不实(应收多/存货差) |
| 1-2 | PB > 3.0, 或高负债, 或无股息, 或连续亏损 |
| 0 | 负资产、资不抵债、严重亏损 |

**塔勒布评分 TS（0-10）** — 不对称性 + 反脆弱性

| 分数 | 标准 |
|------|------|
| 9-10 | 正不对称 > 3:1, 非脆弱/反脆弱, 管理层大量持股, 高风险低 |
| 7-8 | 正不对称 > 2:1, 低脆弱性, 现金流覆盖债务 |
| 5-6 | 中性不对称(1:1-2:1), 中等脆弱性 |
| 3-4 | 负不对称, 脆弱(对单一变量敏感), 管理层激励不合 |
| 1-2 | 严重负不对称, 高脆弱, 杠杆高, 尾部风险大 |
| 0 | 推算了会死的生意(资不抵债、现金流为负、行业消失) |

### 综合得分计算

**Final Score = round((MS × 4 + TS × 3 + SS × 3) / 10, 2)**

权重：芒格 4 : 塔勒布 3 : 施洛斯 3。

芒格权重最高，因为生意质量是长期持有的第一前提；塔勒布和施洛斯等权，作为风险和资产价值的校验。

得分范围 0-10。

**得分解读**：
- **8.0+**：三人共识的优质投资。全面满足质量、安全、不对称标准。
- **7.0-7.9**：很好的投资，但至少一人有保留意见。
- **6.0-6.9**：尚可。通常是一个维度很强，其他维度有缺陷。适合小仓位或观察。
- **5.0-5.9**：平庸。需要进一步研究或放弃。
- **< 5.0**：回避。至少两个维度有明显问题。

## 第八步：方法论守则

1. **数据优先**：未经数据验证的观点不成立。如有缺口必须深挖补齐。
2. **毛利率趋势是第一个信号**：5年毛利率持续下降 = 行业结构性问题。
3. **ROE是一切回报的根源**：ROE < 8%的公司很难是好投资。
4. **负债是第一过滤器**（施洛斯规则）：高负债公司无论多便宜都不碰。
5. **PB是第二过滤器**（施洛斯规则）：PB > 3的股票需要极强的理由才能买。
6. **区分周期vs结构**：利润下降是周期（可恢复）还是结构（不可恢复）？
7. **价格vs价值**（三人交叉验证）：芒格问生意值不值，施洛斯问资产够不够，塔勒布问赔率好不好。
8. **诚实标注Too Hard**：不理解的生意不能投。三人中的任何一人说"I don't know"都应被认真对待。
9. **深挖到根**：对任何"看起来奇怪"的数据点，必须用websearch追查到底。
10. **意见可以不同，数据不能撒谎**：三人可以持不同意见，但所有人都基于同一组事实数据。
